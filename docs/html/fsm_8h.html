◆ fsm_stop_state()

return_codes_t fsm_stop_state ( void  )

Behaviour fot the stop_state. Delete all orders, ignore new orders and stay in stop state while order_button is pressed. If stops on floor, keep door open while in this state. When stop button released, return a return code to use in the lookup_transitions function.

Returns
